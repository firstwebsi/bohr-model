if we see the first rule of Bohr and Bury scheme we get the formula 2n2 by which we can easily think of maximum number of electrons in an atom in each shell

  • in the first shell n=1 ,2n2 2(1x1) = 2 electrons
  • in the second shell n=2 ,2n2 2(2x2) = 8 electrons
